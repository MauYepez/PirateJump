---

## Game Objective

The rules of the sea are simple but challenging:
1. **Explore** the treacherous island layout.
2. **Collect all 3 Gems** scattered across the map.
3. **Reach the Pirate Flag** to secure your treasure and win the game!

*Note: The victory flag will only activate once all 3 gems have been successfully collected.*
